# Step 1: AWSアカウント作成ガイド

## 所要時間: 10-15分

## 必要なもの
- ✅ メールアドレス
- ✅ 電話番号（SMS認証用）
- ✅ クレジットカード（無料枠利用でも登録必須、超過時のみ課金）

---

## 手順

### 1. AWSサインアップページにアクセス
https://aws.amazon.com/

右上の「無料でサインアップ」または「今すぐ無料アカウントを作成」をクリック

### 2. ルートユーザーのメールアドレス入力
- **ルートユーザーのメールアドレス**: あなたのメールアドレス
- **AWSアカウント名**: `mgp-production` など（後で変更可能）
- 「メールアドレスを確認」をクリック

### 3. 認証コード入力
- メールに送られた認証コードを入力
- 「認証を確認」をクリック

### 4. ルートユーザーパスワード設定
- 強力なパスワードを設定（8文字以上、大小英字・数字・記号を含む）
- パスワードをメモしておく（**重要！**）

### 5. 連絡先情報の入力
- **アカウントの種類**: 「個人」を選択
- **フルネーム**: あなたの名前
- **電話番号**: 日本の電話番号（+81から始まる）
- **国/地域**: 日本
- **住所**: あなたの住所
- AWS カスタマーアグリーメントに同意
- 「続行」をクリック

### 6. 支払い情報の入力
- **クレジットカード番号**: カード番号を入力
- **有効期限**: MM/YY形式
- **カード名義人**: カードに記載されている名前
- **請求先住所**: 連絡先情報と同じを選択

**重要**: 
- 無料枠の範囲内では請求されません
- 万が一無料枠を超えた場合のみ課金されます
- 請求アラートを設定することを推奨（後述）

### 7. 本人確認（電話/SMS）
- **確認方法**: 「テキストメッセージ（SMS）」または「音声通話」を選択
- **国コード**: 日本 (+81)
- **電話番号**: あなたの携帯電話番号（最初の0を除く）
  - 例: 090-1234-5678 → 90-1234-5678
- **セキュリティチェック**: 画像の文字を入力
- 「SMSを送信」をクリック

### 8. 認証コード入力
- SMSで受け取った4桁のコードを入力
- 「コードを検証」をクリック

### 9. サポートプランの選択
- **ベーシックサポート - 無料**を選択
- 「サインアップを完了」をクリック

### 10. アカウント作成完了！
- 「AWSマネジメントコンソール」へ移動
- おめでとうございます！AWSアカウントが作成されました 🎉

---

## アカウント作成後の重要な設定

### A. 請求アラートの設定（推奨）

#### 目的: 予期せぬ課金を防ぐ

1. AWSマネジメントコンソール右上のアカウント名 → 「請求ダッシュボード」
2. 左メニュー → 「Billing preferences」
3. 「Receive Free Tier Usage Alerts」をON
4. メールアドレスを入力
5. 「Save preferences」

#### 予算アラートの設定
1. 左メニュー → 「Budgets」
2. 「Create budget」
3. テンプレート: 「Zero spend budget」を選択
   - これで$0.01でも課金があったらアラートが来る
4. 予算名: `Free-Tier-Monitor`
5. メールアドレス入力
6. 「Create budget」

### B. MFA（多要素認証）の設定（推奨）

#### 目的: セキュリティ強化

1. 右上のアカウント名 → 「Security credentials」
2. 「Multi-factor authentication (MFA)」
3. 「Activate MFA」
4. 「Virtual MFA device」を選択
5. スマホアプリ（Google Authenticator等）でQRコードをスキャン
6. 連続する2つのコードを入力

## 完了確認

✅ 以下が完了していることを確認:
- [-] AWSアカウント作成完了
- [-] AWSマネジメントコンソールにログインできる
- [-] 請求アラート設定完了
- [ ] リージョンが「東京 (ap-northeast-1)」になっている
  - 右上のリージョン選択で「アジアパシフィック（東京）」を選択
