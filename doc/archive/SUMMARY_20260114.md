# Mix Poker Game Platform - 進捗サマリー (2026-01-14)

## 📌 プロジェクト概要
ミックスポーカー（NLH, PLO, 2-7TD, Stud等）を多人数でプレイできるオンラインプラットフォームの開発。

## ✅ 完了したマイルストーン

### Phase 1: サーバー基盤構築
- Node.js + Express + Socket.IO によるリアルタイムサーバーの構築。
- TypeScriptによる型安全な実装。

### Phase 2: チップ管理とAWSデプロイ（準備）
- プレイヤーの持ちチップ管理、入退室フローの実装。
- AWS (Amplify, Elastic Beanstalk) へのデプロイ・HTTPS化手順のドキュメント化。

### Phase 3: ゲームエンジン完全実装 (最重要)
- **FSM (有限状態マシン) によるゲームループ**: PREFLOPからSHOWDOWNまでの一貫したフロー。
- **RoomManager**: 6人テーブル、オープン卓/プライベート卓の管理。
- **Game Variants (8種類)**:
  - NLH, PLO, PLO8, 2-7TD, 7CS, 7CS8, RAZZ, BADUGI
- **PotManager**: サイドポットを含む複雑なチップ計算。
- **Auto-Rotation**: 複数ゲームを自動で回す仕組み（HORSE, 8-Gameプリセット）。
- **Meta Game Features**: 7-2ボーナス、スタンドアップゲーム。

## 📁 主要モジュール

| モジュール | 役割 |
| :--- | :--- |
| `GameEngine.ts` | ゲームの進行（フェーズ遷移）を制御。 |
| `PotManager.ts` | 賭け金の集計とポット（メイン・サイド）の構築。 |
| `ActionValidator.ts` | プレイヤーのアクション（Bet, Raise等）が正しいか判定。 |
| `ShowdownManager.ts` | ハンドの強さを比較し、勝者を決定。 |
| `RotationManager.ts` | ゲームの切り替え（ミックス）を管理。 |
| `MetaGameManager.ts` | 7-2ゲームやスタンドアップ等の特殊ルールを管理。 |
